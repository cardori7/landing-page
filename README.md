# landing-page
landing page

Landing page responsiva desenvolvida com HTML5 e CSS3, usando somente dois arquivos: index.html e style.css.
​
Arquivos
index.html: estrutura e conteúdo da página.
​
style.css: estilos, layout e responsividade (media queries/breakpoints).

1. Como rodar
Clone o repositório:
git clone https://github.com/cardori7/landing-page.git

2. Entre na pasta
cd landing-page

3. Abra o index.html no seu navegador
